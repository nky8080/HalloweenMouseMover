Thank you for downloading Halloween Mouse Mover!

WHAT'S INCLUDED:
----------------
This package contains:
- HalloweenMouseMover.exe (Main application)
- Required DLL files (FlaUI, NAudio, and .NET libraries)
- Resource files (horror_mouse.cur, horror_mouse.ico)
- THIRD-PARTY-NOTICES.txt (License information for included libraries)

INSTALLATION:
-------------
1. Extract all files to a folder of your choice
2. Make sure you have .NET 6 Runtime installed
   Download: https://dotnet.microsoft.com/download/dotnet/6.0
3. Run HalloweenMouseMover.exe

AUDIO FILES:
------------
⚠️ IMPORTANT: This package does NOT include Halloween sound effects due to 
copyright restrictions.

To enable sound effects:
1. Prepare your own Halloween sound effect in MP3 format
2. Name it "Scream03.mp3"
3. Place it in the same folder as HalloweenMouseMover.exe

The application will work without the audio file, but sound effects will be 
disabled.

USAGE:
------
- The application runs in the system tray (notification area)
- It automatically detects dialog boxes and moves the cursor to negative buttons
- Right-click the tray icon to access Settings or Exit

SYSTEM REQUIREMENTS:
--------------------
- Windows 10 or later
- .NET 6 Runtime
- Administrator privileges may be required for some features
