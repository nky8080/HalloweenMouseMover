# Product Overview

Halloween Mouse Mover is a Windows desktop application that automatically moves the mouse cursor to negative buttons (No, Cancel, Close) when dialog boxes appear. The application runs in the system tray and provides a Halloween-themed experience with spooky sound effects and cursor transformations.

## Core Functionality

- Monitors for dialog windows in real-time using UI Automation
- Intelligently identifies negative action buttons (No, Cancel, Close, etc.)
- Smoothly animates cursor movement to the negative button with curved trajectories
- Plays Halloween sound effects and changes cursor to horror theme during activation
- Runs as a lightweight background process with system tray integration

## Target Platform

Windows 10 or later, designed for entertainment purposes. The application is inspired by the classic Japanese "Chu-Chu Mouse" application.
